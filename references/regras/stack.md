# 🧱 Stack Obrigatória (React + TypeScript — padrão Lovable)

| Camada            | Tecnologia                              |
|-------------------|-----------------------------------------|
| Framework         | React 18 + TypeScript (strict mode)     |
| Build             | Vite                                    |
| Estilo            | Tailwind CSS + shadcn/ui                |
| Roteamento        | React Router v6                         |
| Estado servidor   | TanStack Query (React Query)            |
| Estado global     | Zustand (quando necessário)             |
| Validação         | Zod                                     |
| Formulários       | React Hook Form + Zod                   |
| Banco / Auth      | Supabase                                |
| Testes unitários  | Vitest                                  |
| Testes E2E        | Playwright                              |
| Deploy            | Vercel                                  |
| CDN / Segurança   | Cloudflare                              |

Não introduza dependências fora desta stack sem aprovar com o usuário e documentar a decisão em `docs/ARQUITETURA.md`.
