Instalação

Um único passo.
O Claude faz o resto.

Não precisa de terminal, npm, ou configuração. Manda o link do GitHub pro Claude Code e pede pra ele instalar a skill globalmente. É só isso.

No terminal, no npm, no config needed. Just send the GitHub link to Claude Code and ask him to install the skill globally.

1

Mande esta mensagem pro Claude Code

Abra qualquer projeto no Claude Code e cole a mensagem abaixo:

Instale globalmente a skill empire-vibe-coding a partir deste repositório:
https://github.com/Empire-Business/empire-vibe-coding

O Claude clona o repo, instala a skill em ~/.claude/skills/ e já ativa. Depois é só digitar /empire-vibe-coding em qualquer projeto.

O que você ganha

Tudo que falta no vibe coding padrão.

Sem framework, o Claude esquece o contexto, deixa código morto, ignora segurança. Com empire-vibe-coding, cada ação tem método.

CLAUDE.md automático

Instala um CLAUDE.md padronizado no seu projeto. Se você já tem um, faz merge inteligente — sem apagar nada.

Segurança nativa

Instala e mantém a skill /security-auditor atualizada globalmente. 27 categorias auditadas antes de cada deploy.

Tasks 100% do tempo

Antes de qualquer ação, cria tasks visíveis. Você sempre sabe exatamente o que o Claude está fazendo — e pode pausar ou redirecionar.

Documentação viva

Toda doc vai para docs/. O CLAUDE.md é um índice. Ao remover código, a doc morre junto — zero lixo acumulado.

Auto-atualização

Peça "verifique atualizações" — a skill faz git pull em si mesma e no security-auditor e reporta o que mudou.

Stack definida

React 18 + TypeScript + Vite + Tailwind + shadcn/ui + Supabase + Vercel. Nenhuma decisão de arquitetura fica sem documentar.
