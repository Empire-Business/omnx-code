Chame a skill
sempre que for codar algo novo.

Cada vez que você iniciar uma feature, refatorar ou documentar algo importante, chame /omnx-code antes. Isso garante que o Claude vai seguir as suas regras — e não inventar nada por conta própria.

Quando usar

Chame a skill antes de qualquer ação que importe para o seu app.

/omnx-code nova feature /omnx-code refatorar auth /omnx-code auditar segurança /omnx-code documentar API /omnx-code deletar componente

O que a skill garante

Antes de codar qualquer coisa, o Claude lê o CLAUDE.md do projeto e cria tasks visíveis. Nenhuma ação ocorre sem que você veja o plano primeiro.

// features

Tudo que um projeto
sério precisa por padrão.

CLAUDE.md automático

Instala ou mescla o template de CLAUDE.md no seu projeto sem destruir o que você já escreveu.

Tasks 100% visíveis

Toda ação é precedida de um plano. O Claude nunca executa nada sem que você veja cada etapa antes.

Security Auditor integrado

Instala e mantém a skill /security-auditor atualizada. Auditorias de RLS, auth e OWASP a um comando.

Sem código morto

Ao remover uma feature, a skill garante que código, docs e imports relacionados também sejam deletados.

Stack validada

React 18 + TypeScript strict + Vite + Tailwind + shadcn/ui + Supabase + Vercel. Nenhuma dependência fora da stack sem aprovação explícita.

Auto-atualização

Peça ao Claude para verificar atualizações — a skill faz git pull em si mesma e no security-auditor automaticamente.
